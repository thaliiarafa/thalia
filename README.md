# Mobile-First Aesthetic Planner & Glow-Up Companion Application

## Overview
This application is designed to help users plan their daily activities while embracing a mobile-first aesthetic. The application aims to enhance the user's life by providing tools for organization, self-improvement, and personal development. 

## Features
- **Aesthetic User Interface:** A beautiful, minimalistic design that provides an engaging user experience.
- **Daily Planner:** Users can easily schedule their tasks and events, ensuring they stay organized.
- **Goal Tracker:** Users can set personal goals and track their progress over time.
- **Self-Care Reminders:** Get reminders for activities that promote mental and physical well-being.
- **Community Sharing:** Share your glow-up journey with a community of like-minded individuals.

## Installation
To get started, clone the repository and install the required dependencies:
```bash
git clone https://github.com/thaliiarafa/thalia.git
cd thalia
npm install
```

## Usage
Run the application locally:
```bash
npm start
```
